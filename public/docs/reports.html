Reports

The Reports interface lists available report definitions, prompts for report parameters, submits report jobs, and displays completed CSV results.

API

  • GET /reports returns the available report ids.
  • GET /reports?include=data returns report definitions keyed by report id.
  • GET /reports/<report_id> returns the selected report definition.
  • POST /reports/<report_id> submits parameter values and returns a job id.
  • GET /jobs/<job_id> returns job status.
  • GET /jobs/files/<job_id>/data.csv returns the completed report CSV.
