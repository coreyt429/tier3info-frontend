Getting Started

Welcome to the Voice Engineering Information Center. Here you will find resources and documentation to help you get started.

Main Page features

The main page layout will be consisten across all pages in the application. It includes:
  • Header
    • Menu toggle button
    • Title: this will change based on the current page
    • Dashboard
      • buttons to represent color states
        • Green: Okay
        • Yellow: Warning
        • Red: Critical
        • Blue: Expired
      • Each button is a dropdown list of dashboard items in its category
      • Clicking a dropdown item will open the corresponding dashboard detail view in a floating modal
    • Current Version
    • Right Drawer toggle button
  • Left Drawer: Contains navigation links
  • Right Drawer: Contains user preferences, scratchpad, and documentation
