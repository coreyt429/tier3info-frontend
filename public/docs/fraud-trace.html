Fraud Trace

The Fraud Trace page searches by IP address or phone number, applies an Elasticsearch-friendly span, and renders the returned JSON keyed by IP address.

Inputs

  • input_string: IP address or phone number.
  • span: Elasticsearch-friendly span, defaulting to 24h.

Endpoint

/fraud/scope/<ip_address|phone_number>/<span>

Displayed Fields

  • Organization and disposition for each IP address.
  • Users registered from the IP address.
  • Password change and last 200 timestamps, with a red indicator when last 200 is newer.
  • Last registration timestamp, code, and device.
