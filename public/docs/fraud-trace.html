Fraud Trace

The Fraud Trace page searches by IP address or phone number, applies an Elasticsearch-friendly span, and renders the returned JSON in tabs for the trace cards, unique IP addresses, and compromised users.

Inputs

  • input_string: IP address or phone number.
  • span: Elasticsearch-friendly span, defaulting to 24h.

Endpoint

/fraud/scope/<ip_address|phone_number>/<span>

Tabs

  • Trace: the current IP address cards with their user tables.
  • IP addresses: one row per unique IP address with its organization.
  • Compromised users: rows for users with a red indicator, including the user table fields plus the IP address that produced the match.
