Broadworks DMS Tag Tool

Getting Started

Welcome to the Broadworks DMS Tag Tool. This tools can be used to set custom tags on multiple device profiles at the same time.

Tagtool features

Tag Input:

  • Text area for entering tags to apply to devices.
  • Tags are added one per line
  • Tag format: %TAG_NAME%=value
  • To delete a tag, set its value to delete
  • Tag input will automatically format some things to prevent errors
  • As you add tags, checkTags will run in the background to update the match status for device profiles

Search Input:

  • A text field for searching devices by various criteria.
  • Supports kibana style searching: field:value
  • Remember to quote values with separators like: group_id:"30000000001-01"

Action Buttons:

  • Rebuild Configs files for the device profiles
  • Reboot Phones
  • Load/Save Tag Sets
    • Load predefined tag sets from a data store.
    • Save current tag sets to a data store for reuse.
    • Tagsets can be managed here
  • Apply Tags:
    • Apply the specified tags to the selected device profiles.
    • Note, this does not automatically rebuild configs or reboot phones

Results Table:

  • A table displaying the search results.
  • Options for selecting and managing devices.
  • Column display can be configured to show/hide specific fields.
  • Filter is a simple string filter, it will match any field in the row
  • Download will save a CSV of the data in the table. Note, this includes rows and fields that are hidden
